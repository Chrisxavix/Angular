# LazyLoad

Proyeto generado en Angular, version 8.3.21.

Las rutas lazyLoad ayudan a que se carguen los componentes de acuerdo al path que el usuario requiere acceder mediante módulos.

En el componente input-add existe inputs que reciben datos del usuario y son visualizados en la pantalla con un **ngIf* de validación.

En el componete admin(padre) recibe datos de suma e interpolación y los visualiza en pantalla, esta comunicación es a través del decorador @Output.
